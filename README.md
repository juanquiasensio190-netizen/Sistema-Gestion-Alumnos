# Sistema de Gestión de Alumnos

## Descripción del Proyecto
Este es un sistema web interactivo desarrollado para la administración académica. Permite registrar alumnos (capturando nombre, documento/cédula y materia), asignarles una nota final en escala del 1 al 5 y gestionar la lista mediante la eliminación dinámica de registros, manteniendo la persistencia de datos local en el navegador mediante LocalStorage.

## Tareas Realizadas
* **Estructura Base (HTML5):** Creación del formulario de registro de datos académicos y maquetación de la tabla de visualización de datos.
* **Diseño e Interfaz (CSS3):** Implementación de un diseño moderno de dos columnas, adaptabilidad responsiva para dispositivos móviles, uso de variables de color y estilos estilizados para botones y tablas.
* **Lógica y Persistencia (JavaScript):** Control dinámico del DOM para la inserción y eliminación de registros, delegación de eventos y manejo de LocalStorage para evitar la pérdida de datos al recargar la página.
